# MONAN-JEDI-WORKFLOW Documentation

MONAN-JEDI-WORKFLOW is an INPE-oriented workflow project for preparing, validating and running MONAN/MPAS-JEDI data assimilation experiments on HPC systems such as JACI.

The documentation is available in two languages:

- [Português](pt/index.md)
- [English](en/index.md)

## Documentation goals

This site separates conceptual documentation, operational usage, developer guidance and practical recipes. It is designed to support both scientific users and workflow developers.

## Current project status

The project currently provides a structured base for a first 3DVar-FGAT experiment on JACI, including environment loading, data layout bootstrap, external input staging, staged input validation, IODA inventory checks, MPAS-JEDI build discovery, template rendering and PBS job rendering.
